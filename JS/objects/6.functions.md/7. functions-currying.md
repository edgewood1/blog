**Two ways of using two parameters**

- Normal

```
const add = (x, y) => x + y 
add(2, 3) /  5

function(x, y) {
  return x + y
}
```

- Curried - here, we pass the arguments distinctly, so that we could add the values from different directions

```
const add = x => y => x + y
add(5)(7)
```
